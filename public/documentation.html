Description

This web application will provide users with loads of information about all of their favorite movies, directors, genres, and more! Not only will your be able to look up your
favorite actors and movies at your leisure, you will be able to personalize your experience. In this application, you will be able to create your own accounte which
will allow you make personalized lists of your go-to movies and which actors make the movie a must-see!

List of URL endpoints and what they do

Business Logic URL HTTP Method Request body data format Response body data format
Return the list of movies /movies GET none JSON object holding the list of movies
Return data about a movie /movies/[title] GET none JSON object holding data about a single movie, example: { title: description: genre: director: actors: imageURL: }
Return movies that fit a certain genre /genres/[genre] GET none JSON object holding a description about a genre
Return info about a director /directors/[director] GET none JSON object holding info on requested director, example: { name: bio: born: death: }
Allow new users to register /users POST JSON object holding data about new user, structured like: { username: password: email: birthday } JSON object confirming success or denial about new user
Searching for user by username users/[username] GET null A JSON object showing the requested user
User updating their info users/[username] PUT JSON object holding data that user wants to update, structured like: { username: password: email: birthday: } A JSON object showing the data that the user updated
Allow Users to add a movie to their favorites list /users/[username]/movies/[movieId] POST null A JSON object showing a list of movies that the user has favorited
User removes a movies from the favorites list /users/[username]/movies/[movieId] DELTE null A confirmation showing that the movie has been removed from the favorites list
User deregistration /users/[username] DELETE null A confirmation showing that the user has successfully deregisted their account
