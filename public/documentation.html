Description

This web application will provide users with loads of information about all of their favorite movies and actors! Not only will your be able to look up your
favorite actors and movies at your leisure, you will be able to personalize your experience. In this application, you will be able to create your own accounte which
will allow you make personalized lists of your go-to movies and which actors make the movie a must-see!

List of URL endpoints and what they do

Business Logic URL HTTP Method Request body data format Response body data format
Return the list of movies /movies GET none JSON object holding the list of movies
Return data about a movie /movies/[title] GET none JSON object holding data about a single movie, example: { title: director: genre: description: }
Return movies that fit a certain genre /movies/genre/[genre] GET none JSON object holding data that list movies that fit a ceratin genre
Return info about a director /movies/[director] GET none JSON object holding info on requested director, example: { name: bio: born: death: }
Allow new users to register /movies/newuser POST JSON object holding data about new user, structured like: { username: password: confirm password: } JSON object confirming success or denial about new user
